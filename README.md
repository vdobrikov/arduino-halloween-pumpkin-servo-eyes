# arduino-halloween-pumpkin-servo-eyes
Arduino based Halloween pumpkin with servo eyes

### MCU:
 - Arduino Nano / Uno

### Servo: 
 - MG 90S Micro xNUM_EYES

### LED strip:
 - WS2812 xNUM_EYES

### Wiring:
```
Arduino       Servo
D2   <----->  Servo1 Signal
...          ...
D7   <----->  Servo6 Signal

Arduino       LED Strip
D12  <----->  Signal
```
